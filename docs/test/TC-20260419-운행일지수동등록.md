## Backend API Tests

### TC-001: 정상 등록 — 필수 필드만

- **사전 조건**: 유효한 JWT, 차량/운전자 등록 완료
- **테스트**: `POST /api/v1/trip-logs` with `{ vehicle_id, driver_id, origin, destination }`
- **기대**: 201, `{ success: true, data: { tripLogId, origin, destination, status: "IN_PROGRESS" } }`
- **검증**: DB에 `car_trip_logs` 1건 + `car_dispatch_requests` 1건 (cdr_is_proxy=1, cdr_note='MANUAL_ENTRY') 생성
- **우선순위**: 상

### TC-002: 정상 등록 — 전체 필드

- **사전 조건**: 유효한 JWT
- **테스트**: `POST /api/v1/trip-logs` with 모든 필드 (vehicle_id, driver_id, origin, destination, depart_actual, arrive_actual, odo_start=1000, odo_end=1050, customer_name, bill_no, cdf_no, refueled=true, fuel_amount=50.5, fuel_cost=1000000, toll_cost=200000, has_accident=false, note, kr_purpose_code=BUSINESS, kr_business_ratio=80)
- **기대**: 201, distance 자동 계산 (1050-1000=50km), 모든 필드 저장
- **우선순위**: 상

### TC-003: 거리 자동 계산

- **테스트**: odo_start=5000, odo_end=5120
- **기대**: `ctlDistanceKm = 120`
- **우선순위**: 상

### TC-004: 거리 미계산 — odo 미입력

- **테스트**: odo_start/odo_end 생략
- **기대**: `ctlDistanceKm = null`
- **우선순위**: 중

### TC-005: Proxy Dispatch 생성 확인

- **테스트**: 수동 등록 후 생성된 dispatch 조회
- **기대**: `cdr_is_proxy=1`, `cdr_status=COMPLETED`, `cdr_note=MANUAL_ENTRY`, `cdr_driver_override=1`
- **우선순위**: 상

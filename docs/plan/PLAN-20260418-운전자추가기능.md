### Phase 2: DriverFormModal 컴포넌트 신규 생성

**Step 2-1: `DriverFormModal.tsx` 구현**

- 위치: `src/components/driver/DriverFormModal.tsx`
- 공용 `Modal` 컴포넌트 재사용
- React Hook Form + Zod 검증
- `useCreateDriver()` mutation 호출
- Props: `{ open, onClose, vehicleId? }`
  - `vehicleId` 전달 시 → 해당 차량에 즉시 배정
  - `vehicleId` 미전달 시 → 차량 미배정 상태로 등록

폼 필드:
| 필드 | 타입 | 검증 | API 필드명 |
|------|------|------|-----------|
| AMA 사용자 ID | text | UUID 형식 필수 | `ama_user_id` |
| 역할 | select | 필수 선택 | `role` |
| 메모 | textarea | 선택 | `note` |

└─ 사이드 임팩트: 없음 (신규 파일)

## 3. Architecture (아키텍처)

### 3.1 Backend Architecture

```
app.module.ts (Root)
├── AuthModule (JWT SSO, Guards, Decorators)
│   ├── JwtStrategy, JwtAuthGuard
│   ├── EntityScopeGuard, RoleGuard
│   └── @Auth(), @CurrentUser(), @Public(), @Roles()
├── SpuMasterModule (Full CRUD)
│   ├── SpuMasterController → SpuMasterService
│   ├── SpuMasterEntity (drd_spu_masters)
│   ├── DTOs: Create/Update/Response
│   └── SpuMasterMapper
├── SkuMasterModule (Full CRUD + Auto Cost History)
│   ├── SkuMasterController → SkuMasterService
│   ├── SkuMasterEntity (drd_sku_masters)
│   ├── DTOs: Create/Update/Response
│   └── SkuMasterMapper
├── ChannelMasterModule (Read Only)
│   ├── ChannelMasterController → ChannelMasterService
│   └── ChannelMasterEntity (drd_channel_masters)
├── ChannelProductMappingModule (Full CRUD)
│   ├── ChannelProductMappingController → ChannelProductMappingService
│   ├── ChannelProductMappingEntity (drd_channel_product_mappings)
│   └── ChannelProductMappingMapper
├── SkuCostHistoryModule (Read Only)
│   ├── SkuCostHistoryController → SkuCostHistoryService
│   └── SkuCostHistoryEntity (drd_sku_cost_histories)
├── HealthController (/api/v1/health)
└── GlobalExceptionFilter
```

### 3.2 Frontend Architecture

```
App.tsx (AmaTokenHandler + Routing)
├── AppLayout.tsx (Blue theme, 5 nav sections)
│   ├── /dashboard → DashboardPage (placeholder)
│   ├── /spu → SpuMasterListPage (table + modal CRUD)
│   ├── /sku → SkuMasterListPage (table + modal + CostHistoryDrawer)
│   └── /channel-mapping → ChannelMappingListPage (table + modal)
├── Auth Pages
│   ├── /login → LoginPage
│   ├── /entity-info → EntityInfoPage
│   └── /change-password → ChangePasswordPage
├── Libs: api-client.ts (drd_token), ama-token.ts, query-client.ts
├── Stores: auth.store.ts, toast.store.ts
├── Services: sales.service.ts (all API client functions)
├── Hooks: useSales.ts (React Query hooks for all domains)
└── i18n: ko/en/vi JSON files (sales namespace)
```

## 4. DB Migration (마이그레이션)

스테이징/프로덕션 배포 시 아래 SQL을 수동 실행해야 한다:

```sql
-- 1. car_vehicle_managers: ent_id 추가
ALTER TABLE car_vehicle_managers
  ADD COLUMN ent_id CHAR(36) NOT NULL AFTER cvm_id;

-- 2. car_import_logs: soft delete 추가
ALTER TABLE car_import_logs
  ADD COLUMN cil_deleted_at DATETIME NULL DEFAULT NULL;

-- 3. car_trip_log_fees: updated_at, soft delete 추가
ALTER TABLE car_trip_log_fees
  ADD COLUMN ctlf_updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  ADD COLUMN ctlf_deleted_at DATETIME NULL DEFAULT NULL;
```

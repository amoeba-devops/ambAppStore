## 1. Common UI Guidelines (공통 UI 지침)

### 1.1 Layout Structure (레이아웃 구조)

모든 앱은 공통 레이아웃을 따른다:

```
┌─────────────────────────────────────────────────────────┐
│ HEADER  (h-14, bg-white, shadow-sm)                     │
│  [앱 로고/이름]        [앱 내 메뉴]    [사용자 정보]      │
├──────────┬──────────────────────────────────────────────┤
│ SIDEBAR  │ CONTENT AREA                                 │
│ (w-60)   │  (flex-1, p-6, bg-gray-50)                  │
│          │                                              │
│ 메뉴 목록 │  페이지 컨텐츠                               │
│          │                                              │
└──────────┴──────────────────────────────────────────────┘
```

- 모바일(< 768px): 사이드바 → 햄버거 드로어 전환
- 컬러 시스템: Amoeba Brand 기준 (`primary: #FF6B2C`, `gray-*` 계열)
- 아이콘: `lucide-react`
- UI 컴포넌트: TailwindCSS + 자체 공통 컴포넌트

### 1.2 공통 컴포넌트 목록

| Component | Description |
|-----------|-------------|
| `<DataTable>` | 페이지네이션, 정렬, 필터 내장 테이블 |
| `<StatusBadge>` | 상태값 뱃지 (색상 자동) |
| `<SearchBar>` | 디바운스 검색 입력 |
| `<DateRangePicker>` | 기간 선택 |
| `<ConfirmModal>` | 삭제/완료 확인 다이얼로그 |
| `<FileUpload>` | 드래그앤드롭 파일 업로드 |
| `<LoadingSpinner>` | 로딩 상태 |
| `<EmptyState>` | 데이터 없음 상태 |
| `<PageHeader>` | 페이지 제목 + 우측 액션 버튼 영역 |

### 1.3 Route 구조 (공통 패턴)

```
/{app-slug}/                    → 대시보드 (홈)
/{app-slug}/login               → 로그인 (필요 시)
/{app-slug}/{resource}          → 목록
/{app-slug}/{resource}/new      → 등록
/{app-slug}/{resource}/:id      → 상세
/{app-slug}/{resource}/:id/edit → 수정
```

## 6. Screen Flow Summary (화면 흐름 요약)

### car-manager 화면 흐름
```
대시보드 → 차량 목록 → 차량 상세
                    → 차량 등록/수정 폼
         → 배차 목록 → 배차 상세(승인/반려)
                    → 배차 신청 폼
         → 배차 캘린더
         → 운행일지 목록 → 운행일지 작성
         → 유지보수 목록 → 유지보수 등록
```

### app-hscode 화면 흐름
```
메인 검색 → 검색 결과 목록 → HS Code 상세 (FTA 세율 탭)
          → AI 분류 화면 → 분류 결과 → HS Code 상세
          → 즐겨찾기 목록 → HS Code 상세
```

### app-sales-report 화면 흐름
```
매출 대시보드 → 채널별 상세 리포트 → Excel 다운로드
              → 데이터 업로드 → 업로드 이력
```

### app-stock-management 화면 흐름
```
재고 현황 대시보드 → 상품 상세 → AI 예측 → 발주서 생성
                  → 입출고 처리
                  → 발주 추천 → 발주서 작성 → 발주서 목록
                  → 발주 목록 → 발주 상세 → 입고 처리
```
